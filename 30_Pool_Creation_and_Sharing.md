# Pool Creation and Sharing

## Overview

This guide covers the complete process of creating pools and sharing them with other organizations. As an issuer, you'll learn how to create pools, add loans, configure sharing options, and share pools with market makers, investors, and other parties.

## Who Can Use This

- Issuers who create and manage pools

## When This Is Used

Use pool creation and sharing when:
- You want to create a new pool
- You need to group loans together
- You want to share pools with other parties
- You need to present loans to investors or market makers

## Step-by-Step Process

### Creating a Pool

1. **Enter Basic Information**
   - Provide a unique pool name
   - Select the asset class
   - Choose the transaction type
   - Add description if needed
   - Indicate if this is a closing deal

2. **Assign Organizations**
   - Select market makers for structuring
   - Choose investors for opportunities
   - Assign servicers if needed
   - Add other parties as required

3. **Create Pool**
   - Review all information
   - Click create button
   - Pool is created and ready for loans

### Adding Loans to Pool

1. **Map Loans**
   - Navigate to pool details
   - Go to loans section
   - Select loans to add
   - Confirm mapping

2. **Review Metrics**
   - Check pool metrics update automatically
   - Verify loan count is correct
   - Review total balance
   - Check weighted averages

### Sharing the Pool

1. **Configure Sharing**
   - Select organizations to share with
   - Set sharing permissions
   - Allow or restrict feedback
   - Allow or restrict downloads

2. **Share Pool**
   - Complete sharing configuration
   - Recipients receive notifications
   - Pool becomes visible to shared parties
   - Recipients can view and review

## Rules & Validations

- Pool names must be unique
- Loans can only belong to one pool at a time
- You can share pools with multiple organizations
- Sharing permissions control what recipients can do
- Pool metrics calculate automatically from mapped loans

## What Happens Next

After creating and sharing a pool:
- Recipients can view and analyze the pool
- Market makers can accept mandates
- Investors can review opportunities
- You receive feedback and can respond
- Pool progresses through workflow stages

Understanding pool creation and sharing helps you effectively present your loan portfolios and collaborate with other parties in structured finance transactions.
