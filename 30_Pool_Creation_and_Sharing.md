# Pool Creation & Sharing

## Overview

This guide covers the complete process of creating pools and sharing them with other organizations. As an issuer, you'll learn how to create pools, add loans, configure sharing options, and share pools with market makers, investors, and other parties.

## Creating a Pool

### Step 1: Access Pool Creation

**How to start:**
1. Navigate to Pools section
2. Click "Create Pool" button
3. Pool creation form opens
4. Ready to enter pool information

### Step 2: Enter Basic Information

**Required information:**
- Pool Name: Unique name for your pool
- Asset Class: Type of loans (Auto Loans, Commercial Mortgage, etc.)
- Transaction Type: Whole Loan, Securitization, Credit Facility, or Participation Agreement
- Description: Additional details about the pool
- Closing Deal: Whether this is a closing deal

**Tips:**
- Use descriptive pool names
- Select correct asset class
- Choose appropriate transaction type
- Provide clear description

### Step 3: Assign Organizations

**You can assign:**
- Market Makers: For structuring and underwriting
- Investors: For investment opportunities
- Servicers: For loan servicing
- Paying Agents: For payment processing
- Rating Agencies: For pool ratings
- Verification Agents: For loan verification

**How to assign:**
- Select organizations from dropdowns
- Can select multiple organizations
- Assign based on your needs
- Can update later if needed

### Step 4: Create Pool

**Process:**
1. Review all information
2. Verify details are correct
3. Click "Create" button
4. Pool is created with "Created" status
5. Ready for loan mapping

---

## Adding Loans to Pool

### Step 1: Map Loans

**How to map:**
1. Navigate to pool details
2. Go to Loans section
3. Click "Map Loans" or "Add Loans"
4. Select loans to add
5. Confirm mapping

### Step 2: Verify Mapping

**After mapping:**
- Loans appear in pool
- Pool metrics update automatically
- Loan count increases
- Balance recalculates
- Status shows pool name

### Step 3: Review Metrics

**Check:**
- Total balance updated
- Loan count correct
- Weighted averages calculated
- Metrics make sense
- Pool ready for sharing

---

## Configuring Sharing Options

### Step 1: Access Sharing Settings

**How to access:**
1. Navigate to pool details
2. Go to Sharing section
3. Click "Configure Sharing" or "Share Options"
4. Sharing configuration opens

### Step 2: Set Permissions

**Configure:**
- View access: What recipients can see
- Download access: Whether they can download data
- Comment access: Whether they can provide feedback
- Action permissions: What actions they can take

### Step 3: Save Configuration

**Process:**
1. Set desired permissions
2. Configure for each recipient type
3. Save sharing configuration
4. Ready to share pool

---

## Sharing the Pool

### Step 1: Select Recipients

**Choose who to share with:**
- Market makers (for mandates)
- Investors (for opportunities)
- Rating agencies (for ratings)
- Other relevant parties

**How to select:**
- Choose from organization list
- Select specific organizations
- Can share with multiple parties
- Can add more later

### Step 2: Share Pool

**Process:**
1. Select recipients
2. Choose sharing permissions
3. Click "Share" or "Share Preview"
4. Pool status may change to "Preview"
5. Recipients receive notifications

### Step 3: Monitor Sharing

**After sharing:**
- Recipients can view pool
- You can see who has viewed
- Track sharing activity
- Monitor feedback

## Troubleshooting

### "Can't create pool"
- Check required fields are filled
- Verify pool name is unique
- Ensure you have permission
- Check for validation errors

### "Can't map loans"
- Verify loans are available
- Check loans aren't already mapped
- Ensure pool status allows mapping
- Verify you have permission

### "Can't share pool"
- Check pool status allows sharing
- Verify you have permission
- Ensure recipients are selected
- Check sharing configuration
