# Token Generation and E-Sign

## Overview

This guide covers how facility agents generate tokens for funding notices and complete the e-signature process for each lender. Learn how to create tokens, sign funding notices, and track per-lender e-signature status.

## Who Can Use This

- Facility Agents who manage funding notices

## When This Is Used

Use token generation and e-sign when:
- Funding request has been approved
- Funding notice needs tokens generated
- You need to sign funding notices for lenders
- You want to complete the drawdown process

## Step-by-Step Process

### Generating Tokens

1. **Access Funding Notice**
   - Navigate to funding notice
   - Verify status shows tokens pending
   - Open token distribution section

2. **Configure Token Distribution**
   - Specify token allocation for each lender
   - Set distribution amounts
   - Confirm allocation percentages
   - Review distribution details

3. **Generate Tokens**
   - Click generate tokens button
   - Confirm token generation
   - Tokens are created for borrower
   - Status updates to show tokens generated

### Completing E-Signatures

1. **Sign for Each Lender**
   - Select first lender
   - Initiate electronic signature
   - Review funding notice
   - Sign electronically for this lender
   - Complete signing process

2. **Repeat for Remaining Lenders**
   - Select next lender
   - Initiate signature process
   - Sign for this lender
   - Continue until all lenders signed

3. **Track Signing Status**
   - View individual lender signing status
   - See which lenders are signed
   - Monitor overall completion
   - Verify all signatures complete

## Rules & Validations

- Tokens must be generated before signing
- You sign for each lender individually
- Each lender's signature is tracked separately
- All lenders must be signed before borrower approval
- Signing status is tracked per lender

## What Happens Next

After generating tokens and completing signatures:
- Tokens are created and allocated
- All lender signatures are complete
- Borrower can approve token transfer
- Funding notice becomes visible to lenders
- Drawdown process can proceed

Understanding token generation and e-sign helps facility agents effectively manage funding notices and complete the drawdown process.
