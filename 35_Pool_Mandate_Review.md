# Pool Mandate Review

## Overview

Pool mandate review is the process where market makers review pools shared with them and decide whether to accept mandates to structure and manage those pools. This guide covers how market makers evaluate pools and make mandate decisions.

## Receiving Pool Mandates

### How Mandates Are Received

**When you receive a mandate:**
- Issuer shares pool with you
- Pool appears in your review queue
- You receive notification
- Pool status may be Preview or Mandate Pending

### What You See

**Pool information:**
- Pool details and metrics
- Loan list and characteristics
- Aggregate statistics
- Supporting documentation
- Issuer information

---

## Reviewing Pools

### Step 1: Access Pool

**How to access:**
1. Check notifications or review queue
2. Navigate to shared pools
3. Open pool details
4. Review pool information

### Step 2: Analyze Pool

**What to evaluate:**
- Pool metrics: Balance, WAC, FICO, LTV, DTI/DSCR
- Loan quality: Individual loan characteristics
- Pool composition: Loan types and distribution
- Risk assessment: Overall risk profile
- Marketability: Attractiveness to investors

### Step 3: Review Documentation

**Check:**
- Pool documentation completeness
- Loan data quality
- Supporting materials
- Compliance information
- Other relevant documents

---

## Making Mandate Decisions

### Option 1: Accept Mandate

**When to accept:**
- Pool meets your criteria
- You want to structure the deal
- Pool is marketable
- Terms are acceptable
- Ready to proceed

**How to accept:**
1. Review pool thoroughly
2. Evaluate pool quality
3. Click "Accept Mandate" button
4. Confirm acceptance
5. Pool status changes accordingly

**What happens:**
- You become the market maker for this pool
- Pool status may change to Mandate Pending or Deal
- You can now structure the deal
- Pool moves forward in workflow

### Option 2: Reject Mandate

**When to reject:**
- Pool doesn't meet your criteria
- Not interested in structuring
- Pool quality concerns
- Terms not acceptable
- Cannot proceed

**How to reject:**
1. Review pool
2. Decide not to accept
3. Click "Reject" button
4. Provide rejection reason (optional)
5. Confirm rejection

**What happens:**
- Mandate rejected
- Pool may return to Preview status
- Issuer can share with other market makers
- You're no longer involved

### Option 3: Request Changes

**When to request changes:**
- Pool has potential but needs improvement
- Issues can be fixed
- Want to work with issuer
- Can proceed after changes

**How to request:**
1. Review pool
2. Identify needed changes
3. Click "Request Changes" or provide feedback
4. Specify what needs to change
5. Submit change request

**What happens:**
- Issuer receives feedback
- Pool may become editable
- Issuer can make changes
- You can review again

## Best Practices

### For Market Makers

**Review process:**
1. **Review thoroughly**: Analyze pool completely
2. **Evaluate metrics**: Check all key statistics
3. **Assess quality**: Evaluate loan characteristics
4. **Consider marketability**: Assess investor appeal
5. **Make informed decision**: Accept, reject, or request changes

**Decision making:**
1. **Be clear**: Provide specific feedback
2. **Be timely**: Respond promptly
3. **Be constructive**: Helpful feedback if rejecting
4. **Be professional**: Maintain good communication

## Common Scenarios

### Scenario 1: Accept Immediately

1. Receive pool mandate → Review
2. Pool meets criteria → Accept
3. Mandate accepted → Structure deal
4. Pool moves forward → Deal creation

### Scenario 2: Request Changes

1. Receive pool mandate → Review
2. Pool needs improvement → Request changes
3. Issuer makes changes → Review again
4. Pool improved → Accept
5. Mandate accepted → Structure deal

### Scenario 3: Rejection

1. Receive pool mandate → Review
2. Pool doesn't meet criteria → Reject
3. Provide feedback → Issuer informed
4. Issuer can share with others → Alternative market makers

## Troubleshooting

### "Can't see pool mandate"
- Check notifications
- Verify pool was shared with you
- Check review queue
- Ensure you're logged in as market maker

### "Can't accept/reject"
- Check pool status allows action
- Verify you have permission
- Ensure mandate is active
- Check if already decided

---

## Key Takeaways

1. **Review thoroughly**: Analyze pool completely before deciding
2. **Three options**: Accept, reject, or request changes
3. **Clear communication**: Provide specific feedback
4. **Timely response**: Don't delay decisions
5. **Professional approach**: Maintain good relationships

Understanding pool mandate review helps market makers effectively evaluate opportunities and make informed decisions about structuring deals.
