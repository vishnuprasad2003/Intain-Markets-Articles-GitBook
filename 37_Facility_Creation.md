# Facility Creation

## Overview

Facility creation is the process where facility agents configure master commitments after they're automatically created from approved term sheets. This guide covers how to set up facility rules, configure lender groups, and complete facility setup.

## Who Can Use This

- Facility Agents who configure master commitments

## When This Is Used

Use facility creation when:
- Term sheet has been approved
- Master commitment is automatically created
- You need to configure facility rules
- You want to set up lender groups

## Step-by-Step Process

### Configuring Facility Rules

1. **Set Up Borrowing Base**
   - Navigate to master commitment
   - Go to Borrowing Base section
   - Set calculation method
   - Define collateral eligibility
   - Set advance rates and limits

2. **Define Collateral Rules**
   - Go to Collateral Rules section
   - Set eligibility criteria
   - Define valuation methods
   - Set concentration limits
   - Configure monitoring requirements

3. **Configure Facility Parameters**
   - Go to Facility Parameters section
   - Set operational parameters
   - Configure reporting requirements
   - Set up covenant templates
   - Complete configuration

### Setting Up Lender Groups

1. **Add Lenders**
   - Navigate to Lender Groups section
   - Click add lender button
   - Select lender organization
   - Enter lender details
   - Add lender to group

2. **Configure Lender Details**
   - Set commitment amount for each lender
   - Configure voting percentages
   - Set lender-specific terms
   - Complete lender configuration

3. **Submit for Approval**
   - Complete all required fields
   - Review configuration
   - Submit for lender approval
   - Status changes to pending approval

## Rules & Validations

- You must complete all required fields before submission
- At least one lender must be configured
- Facility rules must be complete
- Validation must pass before submission
- Once submitted, editing is restricted

## What Happens Next

After configuring master commitment:
- You submit for lender approval
- Lenders receive notifications
- Any lender approval activates the facility
- Facility becomes active when approved
- Borrowers can create funding requests

Understanding facility creation helps facility agents effectively configure master commitments and set up credit facilities for operation.
