# Common User Questions

## Overview

This guide answers common questions users have about pools, loans, and general platform usage. These FAQs help you quickly find answers to typical questions.

## Who Can Use This

- All platform users

## When This Is Used

Use this guide when:
- You have questions about platform features
- You need quick answers to common questions
- You want to understand how things work
- You're looking for troubleshooting help

## Step-by-Step Process

### Pool Questions

1. **Can I edit a pool after sharing it?**
   - Yes, you can usually edit pools in Preview status
   - Once a pool reaches Deal status, editing is typically restricted
   - Check the pool status to see what editing options are available

2. **How many loans can I add to a pool?**
   - There's no specific limit on the number of loans
   - Add as many loans as needed for your transaction
   - The system will calculate metrics automatically

3. **What happens if I remove a loan from a pool?**
   - Loan is excluded from pool calculations
   - Pool metrics update automatically
   - Loan remains visible but marked as removed
   - You can reinstate it later if needed

### Loan Questions

1. **Can a loan be in multiple pools?**
   - No, a loan belongs to one pool at a time
   - To move a loan to another pool, you must first remove it from the current pool

2. **What's the difference between unmapped and mapped loans?**
   - Unmapped: Loan exists in system but not assigned to any pool
   - Mapped: Loan is assigned to a specific pool and included in pool calculations

3. **Can I delete a loan?**
   - You can usually delete unmapped loans that haven't been verified
   - Once a loan is verified or mapped to a pool, deletion may be restricted
   - Check loan status to see deletion options

### General Platform Questions

1. **How do I switch between roles?**
   - Log out and log back in with a different role
   - Each role has its own view and permissions
   - You select your role during login

2. **Why are some buttons disabled?**
   - Wrong status for the action
   - Missing required information
   - Waiting for another party to act
   - Already completed
   - No permission for your role
