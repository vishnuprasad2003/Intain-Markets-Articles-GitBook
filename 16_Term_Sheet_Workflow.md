# Term Sheet Workflow

## Overview

The term sheet is the first step in creating a credit facility. It's the borrower's proposal outlining the key terms of the facility they want. Understanding the term sheet workflow helps you navigate this critical first phase.

## Who Can Use This

- Borrowers who create term sheets
- Facility Agents who review term sheets

## When This Is Used

Use term sheet workflow when:
- You want to propose a new credit facility
- You need to review a borrower's proposal
- You want to understand the approval process

## Step-by-Step Process

### Creating a Term Sheet

1. **Draft Stage**
   - Create term sheet with facility terms
   - Fill in all required information
   - Upload required documents
   - Save your work

2. **Signing Stage**
   - Sign term sheet via electronic signature
   - Review signed version
   - Verify all information is correct

3. **Submission**
   - Submit term sheet for facility agent review
   - Status changes to show it's under review
   - Wait for facility agent decision

4. **Review and Approval**
   - Facility agent reviews your term sheet
   - Facility agent can approve, reject, or request changes
   - If approved, master commitment is automatically created

### Responding to Change Requests

1. **Receive Change Request**
   - Review requested changes
   - Understand what needs to be modified
   - Check any deadlines or priorities

2. **Make Changes**
   - Edit term sheet fields as needed
   - Update documents if required
   - Save your changes

3. **Re-sign if Needed**
   - Complete electronic signature again if required
   - Verify all changes are correct

4. **Resubmit**
   - Resubmit for facility agent review
   - Status changes back to review
   - Process can repeat until approved

## Rules & Validations

- You can only edit term sheets while they're in Draft or Changes Requested status
- You must sign the term sheet before submitting
- Facility agents can approve, reject, or request changes
- Approved term sheets automatically create master commitments
- Rejected term sheets cannot be resubmitted

## What Happens Next

After creating a term sheet:
- You can sign and submit it for review
- Facility agent reviews and makes a decision
- If approved, master commitment is automatically created
- If changes are requested, you can update and resubmit
- If rejected, you can create a new term sheet

Understanding the term sheet workflow helps you effectively navigate the first phase of credit facility creation and know what to expect at each stage.
