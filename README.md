# Intain Markets User Guide Articles

## Article Naming Convention

Articles use underscores instead of spaces in filenames:
- Example: "Pool Lifecycle & Statuses" → `Pool_Lifecycle_and_Statuses.md`
- Special characters: `&` → `and`, `/` → `_`

## Content Guidelines

All articles follow these principles:
- **User-friendly**: Written from UI/portal perspective, not technical
- **Clear explanations**: Easy to understand for typical end users
- **Comprehensive**: Cover all aspects of the topic
- **Practical**: Include examples and common scenarios
- **Well-structured**: Use headers, lists, and clear sections
