# Funding and Approval Questions

## Overview

This guide answers common questions about credit facilities, funding requests, approvals, and the funding process. These FAQs help you understand the funding workflow.

## Who Can Use This

- Borrowers who use credit facilities
- Facility Agents who manage facilities
- Lenders who participate in facilities

## When This Is Used

Use this guide when:
- You have questions about credit facilities
- You need answers about funding requests
- You want to understand approval processes
- You're looking for funding workflow information

## Step-by-Step Process

### Term Sheet Questions

1. **How long does term sheet review take?**
   - Review time varies depending on facility agent workload
   - You'll receive notifications when status changes
   - Check with facility agent if concerned about timing

2. **What happens if my term sheet is rejected?**
   - Term sheet enters a final state
   - No master commitment is created
   - You can create a new term sheet to try again
   - Address the issues that led to rejection

3. **Do I need to re-sign if changes are requested?**
   - Usually yes, when changes are requested
   - The system will indicate if re-signing is needed
   - Complete electronic signature again after making changes

### Master Commitment Questions

1. **Why was a master commitment created automatically?**
   - Master commitments are automatically created when term sheet is approved
   - System creates it with pre-populated information from term sheet
   - Facility agent then configures it

2. **How long does lender approval take?**
   - Lender approval time varies
   - Any lender approval activates the facility
   - You'll be notified when facility becomes active

3. **Can I see which lenders have approved?**
   - Yes, you can typically see lender approval status
   - Each lender's approval status is tracked individually
   - Shows who has approved and when

### Funding Request Questions

1. **How much can I request in a funding request?**
   - You can request up to the available borrowing capacity
   - The system shows available capacity
   - You cannot request more than what's available

2. **Can I create multiple funding requests?**
   - Yes, you can create multiple funding requests
   - As long as you don't exceed total facility limits
   - Each request goes through review process

3. **What happens if my funding request is rejected?**
   - No funding notice is created
   - Drawdown doesn't proceed
   - You can create a new funding request
   - Address issues that led to rejection

### Funding Notice Questions

1. **Why do I need to approve token transfer?**
   - Borrower token approval enables lender visibility
   - Until you approve, lenders cannot see the funding notice
   - Approval authorizes the token transfer

2. **What are tokens?**
   - Tokens represent drawdown amounts
   - They enable tracking and distribution
   - They're allocated to lenders based on commitments
