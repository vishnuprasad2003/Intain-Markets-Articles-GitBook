# Status or Logic Changes

## Overview

This document tracks changes to status workflows, business logic, and process flows. Understanding these changes helps you adapt to platform updates and know what to expect.

## Types of Changes

### Status Workflow Changes

**What changes:**
- Status values
- Status transitions
- Status flow
- Status meanings
- Status rules

**Examples:**
- New status added
- Status transition changed
- Status flow modified
- Status meaning updated
- Status rules revised

---

### Business Logic Changes

**What changes:**
- Approval processes
- Validation rules
- Calculation methods
- Workflow steps
- Business rules

**Examples:**
- Approval criteria changed
- Validation rules updated
- Calculation methods revised
- Workflow steps modified
- Business rules updated
