# Funding Request Review

## Overview

Funding request review is the process where facility agents evaluate borrower funding requests to ensure compliance with facility rules, verify borrowing base availability, and make approval decisions. This guide covers the review process from a facility agent perspective.

## Who Can Use This

- Facility Agents who review funding requests

## When This Is Used

Use funding request review when:
- Borrowers submit funding requests
- You need to evaluate drawdown requests
- You want to approve or reject requests
- You need to ensure facility rules are followed

## Step-by-Step Process

### Reviewing Funding Requests

1. **Access Request**
   - Check notifications or review queue
   - Navigate to funding requests for review
   - Open request details
   - Review all information

2. **Verify Compliance**
   - Check request complies with facility rules
   - Verify amount is within limits
   - Confirm draw frequency compliance
   - Ensure purpose aligns with facility

3. **Check Borrowing Base**
   - Verify sufficient capacity available
   - Assess request impact on borrowing base
   - Check collateral support if applicable
   - Verify calculations are correct

4. **Review Documentation**
   - Check all required documents provided
   - Verify information is accurate
   - Ensure documents support request
   - Confirm documentation quality

5. **Evaluate Collateral** (if applicable)
   - Verify collateral eligibility
   - Check collateral valuation
   - Confirm collateral availability
   - Ensure concentration within limits

### Making Review Decisions

1. **Approve Request**
   - Review thoroughly
   - Verify all requirements met
   - Click approve button
   - Confirm approval
   - Funding notice is automatically created

2. **Reject Request**
   - Review and identify issues
   - Click reject button
   - Provide rejection reason
   - Confirm rejection
   - Borrower can create new request

3. **Request Changes**
   - Identify needed modifications
   - Click request changes button
   - Provide change details
   - Confirm change request
   - Borrower can update and resubmit

## Rules & Validations

- Requests must comply with all facility rules
- Request amount cannot exceed available capacity
- All required documentation must be provided
- Collateral must meet eligibility criteria if applicable
- Approved requests automatically generate funding notices

## What Happens Next

After reviewing a funding request:
- If approved, funding notice is automatically created
- If rejected, borrower receives reason and can create new request
- If changes requested, borrower can update and resubmit
- Status updates reflect your decision
- Borrower is notified of the outcome

Understanding funding request review helps facility agents effectively evaluate requests and ensure facility rules are followed.
