# Loan Management

## Overview

Loan management covers all the tasks issuers perform to manage loans in the platform, including uploading loans, mapping them to pools, updating loan information, and managing loan statuses.

## Who Can Use This

- Issuers who manage loans

## When This Is Used

Use loan management when:
- You need to upload loan data
- You want to map loans to pools
- You need to update loan information
- You want to manage loan statuses

## Step-by-Step Process

### Uploading Loans

1. **Prepare Loan Data**
   - Organize loan data in a file
   - Ensure all required fields are included
   - Verify data accuracy

2. **Upload File**
   - Navigate to Loans section
   - Click upload button
   - Select your loan file
   - Upload to the system

3. **Review Standardized Loans**
   - System processes and standardizes data
   - Review standardized loans
   - Verify information is correct
   - Loans are ready for mapping

### Mapping Loans to Pools

1. **Select Loans**
   - Navigate to loan list
   - Select loans to map
   - Can select multiple loans

2. **Choose Pool**
   - Select target pool from dropdown
   - Verify correct pool selected
   - Confirm mapping

3. **Complete Mapping**
   - Loans are mapped to pool
   - Pool metrics update automatically
   - Loans show pool name as status

### Managing Loan Statuses

1. **View Status**
   - Check loan status in loan list
   - View status on loan details page
   - Understand what status means

2. **Update Status**
   - Change status as needed
   - Remove loans from pools if necessary
   - Reinstate removed loans when appropriate

## Rules & Validations

- Loans can only belong to one pool at a time
- You must unmap a loan from one pool before mapping to another
- Removed loans are excluded from pool calculations
- Some statuses restrict certain actions
- Pool metrics update automatically when loans are mapped or removed

## What Happens Next

After managing loans:
- Loans are available for pool creation
- Pool metrics calculate automatically
- You can track loans through their lifecycle
- Loan status changes affect pool calculations
- You can manage loans throughout their journey

Understanding loan management helps you effectively organize your loan portfolios and create successful pools for structured finance transactions.
