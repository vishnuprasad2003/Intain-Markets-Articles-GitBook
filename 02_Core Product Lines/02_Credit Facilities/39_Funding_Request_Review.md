# Funding Request Review

## Overview

Funding request review is the process where facility agents evaluate borrower funding requests to ensure compliance with facility rules, verify borrowing base availability, and make approval decisions. This guide covers the review process from a facility agent perspective.

## Receiving Funding Requests

### How Requests Are Received

**When you receive a request:**
- Borrower submits funding request
- Status changes to FAReview
- You receive notification
- Request appears in review queue

### What You See

**Request information:**
- Requested amount
- Purpose of funds
- Funding date
- Supporting documentation
- Collateral information (if applicable)
- Borrower information

---

## Review Process

### Step 1: Access Request

**How to access:**
1. Check notifications or review queue
2. Navigate to funding requests for review
3. Open request details
4. Review all information

### Step 2: Verify Compliance

**What to check:**
- Facility rules: Request complies with facility terms
- Amount limits: Within facility and borrowing base limits
- Draw frequency: Meets frequency requirements
- Purpose: Aligns with facility purpose
- Other rules: Meets all facility requirements

### Step 3: Check Borrowing Base

**What to verify:**
- Available capacity: Sufficient borrowing base available
- Request impact: How request affects borrowing base
- Collateral support: Collateral supports request (if applicable)
- Calculations: Borrowing base calculations correct
- Remaining capacity: Capacity after request

### Step 4: Review Documentation

**What to check:**
- Completeness: All required documents provided
- Accuracy: Information is accurate
- Support: Documents support the request
- Quality: Documentation meets requirements

### Step 5: Evaluate Collateral (If Applicable)

**What to evaluate:**
- Eligibility: Collateral meets facility criteria
- Valuation: Collateral properly valued
- Availability: Collateral not already pledged
- Concentration: Within concentration limits

---

## Making Review Decisions

### Option 1: Approve Request

**When to approve:**
- Request meets all facility rules
- Sufficient borrowing capacity available
- Documentation complete and accurate
- Collateral eligible (if applicable)
- No compliance issues

**How to approve:**
1. Review request thoroughly
2. Verify all requirements met
3. Click "Approve" button
4. Confirm approval
5. Status changes to APPROVED

**What happens:**
- Funding request status: APPROVED
- Funding notice automatically generated
- Funding notice status: PENDING_TOKEN_GENERATION
- Ready for token generation
- Borrower notified

### Option 2: Reject Request

**When to reject:**
- Request violates facility rules
- Insufficient borrowing capacity
- Documentation insufficient or inaccurate
- Collateral ineligible (if applicable)
- Cannot be fixed with changes

**How to reject:**
1. Review request
2. Decide to reject
3. Click "Reject" button
4. Provide rejection reason (recommended)
5. Confirm rejection

**What happens:**
- Funding request status: REJECTED
- No funding notice created
- Final state - no further action
- Borrower notified
- Borrower can create new request

### Option 3: Request Changes

**When to request changes:**
- Request has correctable issues
- Missing or incomplete documentation
- Minor compliance issues
- Needs clarification or correction
- Can be fixed with modifications

**How to request:**
1. Review request
2. Identify needed changes
3. Click "Request Changes" button
4. Specify what needs to change
5. Provide change details
6. Submit change request

**What happens:**
- Funding request status: CHANGES_REQUESTED
- Revision number increments
- Current version saved to history
- Borrower can edit and resubmit
- Borrower notified

## Best Practices

### For Facility Agents

**Review process:**
1. **Review thoroughly**: Check all aspects carefully
2. **Verify calculations**: Ensure borrowing base correct
3. **Check compliance**: Verify all rules met
4. **Document decisions**: Provide clear feedback
5. **Respond promptly**: Keep workflow moving

**Decision making:**
1. **Be thorough**: Check all requirements
2. **Be clear**: Provide specific feedback
3. **Be timely**: Respond promptly
4. **Be consistent**: Apply rules consistently
5. **Be professional**: Maintain good communication

## Common Scenarios

### Scenario 1: Perfect Request

1. Receive request → Review
2. All requirements met → Approve
3. Funding notice generated → Proceeds

### Scenario 2: Missing Documentation

1. Receive request → Review
2. Documentation incomplete → Request Changes
3. Borrower adds documents → Resubmits
4. Review again → Approve

### Scenario 3: Exceeds Capacity

1. Receive request → Review
2. Amount exceeds available capacity → Reject
3. Provide rejection reason
4. Borrower can create smaller request

---

## Troubleshooting

### "Can't find request to review"
- Check review queue
- Verify request status is FAReview
- Check filters
- Ensure you're assigned as facility agent

### "Unclear what to check"
- Review facility rules
- Check borrowing base calculations
- Verify documentation requirements
- Consult facility terms

---

## Key Takeaways

1. **Comprehensive review**: Check all compliance aspects
2. **Three decisions**: Approve, reject, or request changes
3. **Automatic actions**: Approval generates funding notice
4. **Clear feedback**: Provide specific guidance
5. **Efficient process**: Keep workflow moving

Understanding funding request review helps facility agents effectively evaluate requests and ensure facility compliance while enabling borrowers to access funds when appropriate.
