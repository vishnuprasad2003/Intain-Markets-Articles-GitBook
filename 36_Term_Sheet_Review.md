# Term Sheet Review

## Overview

Term sheet review is the process where facility agents evaluate term sheets submitted by borrowers and decide whether to approve, reject, or request changes. This guide covers how facility agents review term sheets effectively.

## Receiving Term Sheets for Review

### How Term Sheets Are Received

**When you receive a term sheet:**
- Borrower submits term sheet
- Status changes to FAReview
- You receive notification
- Term sheet appears in review queue

### What You See

**Term sheet information:**
- Facility terms and amounts
- Interest rates and margins
- Maturity dates
- Commercial terms
- Supporting documentation
- Borrower information

---

## Reviewing Term Sheets

### Step 1: Access Term Sheet

**How to access:**
1. Check notifications or review queue
2. Navigate to term sheets for review
3. Open term sheet details
4. Review all information

### Step 2: Evaluate Terms

**What to evaluate:**
- Facility amount: Appropriate for borrower
- Interest rates: Competitive and appropriate
- Terms: Reasonable and acceptable
- Structure: Suitable for facility type
- Risk assessment: Borrower creditworthiness

### Step 3: Review Documentation

**Check:**
- Financial statements completeness
- KYC documents (if required)
- Collateral documentation (if applicable)
- Supporting materials
- Documentation quality

### Step 4: Assess Compliance

**Verify:**
- Terms meet facility requirements
- Borrower qualifies
- Documentation complete
- Compliance with policies
- Risk acceptable

---

## Making Review Decisions

### Option 1: Approve Term Sheet

**When to approve:**
- Terms are acceptable
- Borrower qualifies
- Documentation complete
- Risk acceptable
- Ready to proceed

**How to approve:**
1. Review term sheet thoroughly
2. Verify all requirements met
3. Click "Approve" button
4. Confirm approval
5. Status changes to Accepted

**What happens:**
- Term sheet status: Accepted
- Master commitment automatically created
- Master commitment starts in Draft status
- You can now configure master commitment
- Borrower notified

### Option 2: Reject Term Sheet

**When to reject:**
- Terms not acceptable
- Borrower doesn't qualify
- Cannot be fixed
- Risk too high
- Not viable

**How to reject:**
1. Review term sheet
2. Decide to reject
3. Click "Reject" button
4. Provide rejection reason (recommended)
5. Confirm rejection

**What happens:**
- Term sheet status: Rejected
- No master commitment created
- Final state - no further action
- Borrower notified
- Borrower can create new term sheet

### Option 3: Request Changes

**When to request changes:**
- Terms need adjustment
- Documentation incomplete
- Issues can be fixed
- Want to work with borrower
- Can proceed after changes

**How to request:**
1. Review term sheet
2. Identify needed changes
3. Click "Request Changes" button
4. Specify what needs to change
5. Provide change details
6. Submit change request

**What happens:**
- Term sheet status: CHANGES_REQUESTED
- Revision number increments
- Current version saved to history
- Borrower can edit and resubmit
- Borrower notified

## Best Practices

### For Facility Agents

**Review process:**
1. **Review thoroughly**: Check all aspects carefully
2. **Evaluate terms**: Assess facility terms
3. **Verify documentation**: Check completeness
4. **Assess risk**: Evaluate borrower creditworthiness
5. **Make informed decision**: Approve, reject, or request changes

**Decision making:**
1. **Be clear**: Provide specific feedback
2. **Be timely**: Respond promptly
3. **Be constructive**: Helpful guidance if requesting changes
4. **Be professional**: Maintain good communication

**Communication:**
1. **Explain decisions**: Provide clear reasoning
2. **Specify changes**: Detail what needs to change
3. **Set expectations**: Clarify next steps
4. **Maintain relationships**: Professional approach

## Common Scenarios

### Scenario 1: Immediate Approval

1. Receive term sheet → Review
2. Terms acceptable → Approve
3. Status: Accepted → Master commitment created
4. Configure master commitment → Facility setup

### Scenario 2: Changes Requested

1. Receive term sheet → Review
2. Needs modifications → Request changes
3. Borrower makes changes → Resubmits
4. Review again → Approve
5. Status: Accepted → Master commitment created

### Scenario 3: Rejection

1. Receive term sheet → Review
2. Terms not acceptable → Reject
3. Provide rejection reason
4. Borrower creates new term sheet → Try again

## Troubleshooting

### "Can't see term sheet for review"
- Check notifications
- Verify term sheet status is FAReview
- Check review queue
- Ensure you're assigned as facility agent

### "Can't approve/reject"
- Check term sheet status
- Verify you have permission
- Ensure review is active
- Check if already decided

---

## Key Takeaways

1. **Review thoroughly**: Evaluate all aspects before deciding
2. **Three decisions**: Approve, reject, or request changes
3. **Clear feedback**: Provide specific guidance
4. **Timely response**: Don't delay decisions
5. **Professional approach**: Maintain good relationships

Understanding term sheet review helps facility agents effectively evaluate borrower requests and make informed decisions about credit facilities.
